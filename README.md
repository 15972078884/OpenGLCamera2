## Android OpenGL Camera 2.0 

项目使用 OpenGL ES 3.0 实现 30 种相机滤镜和特效，支持相机预览和拍照，左右滑动切换滤镜。对 OpenGL ES 3.0 不熟悉的同学可以移步[Android OpenGLES 3.0 开发系统性学习教程。](https://github.com/githubhaohao/NDK_OpenGLES_3_0)
